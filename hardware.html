Summary of Networking and Computer Hardware

  • Hardware is the physical component of a computer or its peripheries.
  • The motherboard is the main physical component of the PC. It houses the CPU and is a hub through which all other hardware runs through.

    • Other computer hardware include:

    • CPU - Central Processing Unit
    • RAM - Random Access Memory
    • Hard Drive
    • GPU - Graphics Processing Unit
    • Power Supply Unit
    • Keyboard
    • Mouse

    Networking Hardware

    • Router
    • Switch
    • Hub
    • Ethernet cables
