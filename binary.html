Summary of Binary Numbers

  1. Computers operate on base 2 number system, processing information in terms of 0s and 1s.
  2. A zero or one represents a bit, short for binary digit.
  3. A bit can only be either a 0 for off or a 1 for on. Eight bits make one byte.
  4. Base 2 number systems can be converted to other number sytems like base 10 (decimal number system) or base 16 (hexadecimal number system) and the reverse is also true.
  5. Binary number systems are used in computer applications. All the coding and languages in computers such as C, C++, Java, etc. use binary digits 0 and 1 to write a program or encode any digital data. The computer understands only the coded language. Therefore these 2-digit number system is used to represent a set of data or information in discrete bits of information.
